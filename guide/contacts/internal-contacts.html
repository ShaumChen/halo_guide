# Internal Contacts

# 1. Hierarchy

Can edit departments, add sub departments, sort sub departments, sort employees and view basic information of employees

# 1.1 Modify Department

can modify department name, superior department, department supervisor, create / disband department group, and set group owner

Tip

Top level department cannot modify department name and superior department

# 1.2 Create sub department

Click Create Department ，Fill in the corresponding information and create a sub department

# 1.3 Department Sort

Click sort ，Sort departments

# 1.4 Employee Sort

Staff sort is the same as department sort

# 1.5 View employee basic information

Click the employee，view the employee information（Basic Information、Staff Management、Holiday Management）

The employee's name, phone, Department, role and position can be modified

# 2. Role

Can add a role group、a role, modify the corresponding role information, and add role members

# 2.1 Create Role Group

Click Create Role Group ，fill in the corresponding information and add a new role group

# 2.2 Create Role

Click Create Role ，Fill in the corresponding information and add a new role

# 2.3 Modify Role

Click Modify ，can modify role names and role groups

Tip

The default role does not support editing

# 2.4 Add Members

Click Add Members ，members can be added under corresponding roles
