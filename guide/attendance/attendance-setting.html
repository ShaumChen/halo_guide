# Attendance setting

Attendance group management, shift management, overtime rules, supplementary card rules and leave management can be set

# 1. Attendance group management

Note: Please go to the shift management to add shifts before setting up the attendance group management.

In the attendance group management,You can add attendance group, edit attend scheduling, modify attendance rules and delete attendance group.

# New class rules

Click New class rules ，fill in the information

# Edit scheduling

Click Edit scheduling ，schedule management

Tip

Only when the attendance group type is shift scheduling system, can edit the shift scheduling

# Modify rules

Click Modify rules ，scheduling rules can be modified

# Delete attendance group

Click Delete ，pop up the delete confirmation box and execute the operation

# 2. Shift management

In the shift management, you can add shifts, edit shifts, delete shifts,as shown in the figure below:

# 2.1 New shift

Click the New Shift button at the top left corner of the shift list, and the new shift interface will pop up on the right, as shown in the figure below:

Fill in the information in the new shift interface and click the Confirm button in the lower right corner to add a new shift. The name of the shift, work on/off time, and class owner are required, and the card range limit and rest time are not required. If you fill in the card range limit, you are only allowed to check-in within the time range you filled in, and you cannot check-in in other time periods. If you fill in the rest time, this part of the time will be deducted when calculating the working hours of a certain day.

# 2.2 Editing shift

Click the Edit button on the right side of the shift list, and the edit shift interface will pop up on the right. The shift information can be edited, and the editing content is the same as the add shift content.

# 2.3 Delete shift

Click the Delete button on the right side of the shift list, and a confirmation delete popup will pop up. Clicking confirm will delete this shift information. When the shift information is used by the attendance group, it is not allowed to delete it.

# 2.4 List search query

Click the search icon at the top right corner of the shift list and enter the keyword (shift name) to filter and query.

# 3. Overtime rule

Overtime rules can be added, edited and deleted

# New class rules

Click New class rules ，fill in the information

# Edit overtime rule

Click Edit ，modify overtime rule information

# Delete overtime rule

Click Delete ，Pop up the delete confirmation box and execute the operation

# 4. Card replacement rules

Can add, edit and delete card filling rules

# New add-in card rules

Click New add-in card rules ，fill in the information

# Edit card supplement rules

Click Edit ，modify the information of card replacement rules

# Delete supplementary card rule

Click Delete ，Pop up the delete confirmation box and execute the operation
