# Oa Design/Edit

There are Basic settigns , Form design , Process design

The form cannot be published successfully until the three settings are completed

After publishing successfully, you can view the published form on the mobile terminal

# Basic settigns

  • Can set the Form Icon , Form Name , Form Group , Form description and Form Initiator
  • Click the form icon to select other icon styles or upload other pictures as form icons

# Form design

Form design content：

  • On the left are form controls and form kits. Drag to add form content

  • The middle is the display style of the form, and the content is dragged in from the control library on the left

  • On the right is the information of the currently selected control / suite, which can be modified

Tip

There is only one suit in the form, and it cannot be dragged into the Details/forms control

Details/forms cannot be nested within each other

# Conditional control

Tip

Input numbers 、Amount 、Formula are called a conditional control

If the condition control is required, it will participate in process design as process approval condition

If condition control is used in process design, the corresponding control cannot be deleted in form design

# Process design

  • Set conditions

  • Set approvers

  • Set Cc

# Set Conditions

  • Click the Add button or Add conditions to add the Condition node

  • Click the node name to modify the node name

  • Click the Condition node to open the corresponding condition node information

# Add Initiator

  • Click initiator to select specific person / Department / role

# Add conditions

  • Click Add conditions ，By default, only the initiator condition option is available. If required is checked in condition control, it will be used as process approval condition

# Add condition groups

  • Click Add condition groups ，can be deleted

# Set approvers

  • Click the node name to modify the node name

# Approval node information

  • Click approval node to open the information of corresponding approval node

  • There are six modules: Designated Member , Sponsor's select , Supervisor , Department head , Role , Self Approve

  • By default, select the module of Sponsor's select

# Set Cc

  • Click the node name to modify the node name

  • Click the CC node to open the corresponding CC node information

    Click Add Member ，can add Designated Member , The head of and Master Administrator
