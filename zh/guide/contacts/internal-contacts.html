# 内部通讯录管理

# 1. 组织架构

可编辑部门、添加子部门、子部门排序、员工排序以及查看员工基本信息

# 1.1 编辑部门

可修改部门名字、上级部门、部门主管、创建/解散部门群、设置群主

注意

顶级部门不可修改部门名字和上级部门

# 1.2 创建子部门

点击 Create Department ，填入相应信息创建子部门

# 1.3 部门排序

点击 sort ，进行部门排序

# 1.4 员工排序

员工排序同部门排序

# 1.5 查看员工基本信息

点击员工，页面右侧弹出对应员工信息（基本信息、员工档案、假期管理）

可对员工姓名、手机号、所属部门、角色以及职位进行修改

# 2. 角色

可新增角色组、新增角色、修改对应角色信息、添加角色成员

# 2.1 新增角色组

点击 新增角色组 ，填入相应信息新增角色组

# 2.2 新增角色

点击 新增角色 ，填入相应信息新增角色

# 2.3 编辑角色

点击 编辑 ，可编辑角色名称以及角色分组

注意

默认角色不支持编辑

# 2.4 添加成员

点击 添加成员 ，可在对应角色下添加成员
