# 表单设计/编辑

主要有基础设置、表单设计、流程设计

三项设置完成后才能成功发布表单

发布成功后可在手机端查看已发布表单

# 基础设置

  • 可设置表单图标、名称、所在分组、表单说明、表单发起人
  • 点击表单图标，可选择其他图标样式，也可上传其他图片作为表单图标

# 表单设计

表单设计内容：

  • 左侧为表单控件以及表单套件，拖动添加表单内容

  • 中间为表单的展示样式，内容由左侧控件库拖入

  • 右侧为当前选中控件/套件信息，可进行修改

注意

表单中的套件有且仅有一个，且不可拖入 明细/表格 控件内

明细/表格 控件不能相互嵌套

# 条件控件

注意

数字输入框 、金额 、计算公式 被称为条件控件

条件控件勾选必填后，将作为 流程审批条件 参与 流程设计

若在流程设计中使用到了条件控件，则在 表单设计 中，无法删除对应控件

# 流程设计

  • 设置条件

  • 设置审批人

  • 设置抄送人

# 设置条件

  • 点击添加按钮或者 添加条件 ，即可添加条件节点

  • 点击节点名即可修改节点名字

  • 点击条件节点，打开对应条件节点信息

# 添加发起人

  • 点击发起人，可选择具体人员/部门/角色

# 添加条件

  • 点击 添加条件 ，默认只有 发起人 条件选项。如果 条件控件 勾选了必填，则会作为流程审批条件

# 添加条件组

  • 点击 添加条件组 ，可删除

# 设置审批人

  • 点击节点名即可修改节点名字

# 审批节点信息

  • 点击审批节点，打开对应审批节点信息

  • 有 指定成员 ，发起人自选 ，连续多级主管 ，部门主管 ，角色 ，发起人自己 六个模块

  • 默认选中 发起人自选 模块

# 设置抄送人

  • 点击节点名即可修改节点名字

  • 点击抄送节点，打开对应抄送节点信息

    点击 添加成员 ，可添加 指定成员 、负责人 、主管理员
