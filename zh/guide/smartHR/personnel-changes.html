# 人事异动

可进行入职管理、转正管理、合同管理、离职管理以及人事异动记录（入职记录、转正记录、调岗记录、离职记录）查看

# 1. 入职管理

可对要入职人员进行相应操作，可以办理入职

1.1 办理入职

  • 点击 办理入职 ，进行入职操作

  • 可扫码入职、审批入职、手动添加。默认手动添加入职

# 2. 转正管理

对需要转正的员工进行转正操作

# 3. 合同管理

对员工合同进行操作，可根据部门进行筛选

注意

暂不支持编辑，后续开发正加紧制作ing

# 4. 离职管理

对要离职员工进行离职操作。可根据职员、部门以及时间进行筛选

4.1 办理离职

  • 点击 办理离职 ，进行离职操作

  • 可审批离职、手动办理。默认手动办理离职

# 5. 人事异动记录

可查看入职记录、转正记录、调岗记录、离职记录

# 5.1 入职记录

可根据职员、部门、入职时间进行入职记录筛选

点击 详情 ，可查看对应员工档案信息

# 5.2 转正记录

同入职记录

# 5.3 调岗记录

同入职记录

# 5.4 离职记录

同入职记录
