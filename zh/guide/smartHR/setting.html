# 员工档案字段设置

可新建档案字段组、分组排序、档案组设置、编辑/删除档案组字段

# 1 新建分组

点击 新建分组 ，输入分组名称添加新分组

# 2 分组排序

点击 分组排序 ，调整分组顺序

# 3 档案组设置

点击 设置 ，可新增字段、删除分组、编辑分组、字段调整顺序

3.1 新增字段

点击 新增字段 ，填入信息新增字段

3.2 删除分组

点击 删除分组 ，弹出确认删除弹框，执行相应操作

3.3 编辑分组

点击 编辑分组 ，弹出编辑弹框，可修改分组名称

3.4 分组字段调整

点击 字段调整顺序 ，拖动字段调整顺序

# 4 编辑/删除档案组字段

  • 点击 编辑 ，可编辑档案组字段信息，默认系统字段不可修改

  • 点击 删除 ，弹出确认删除弹框，执行相应操作
