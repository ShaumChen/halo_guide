# 自定义手机版工作台

可创建菜单组、菜单组排序、编辑菜单组（重命名、删除该菜单、新增菜单）、菜单排序、菜单编辑、以及菜单删除

# 1. 创建菜单组

点击 创建菜单组 ，弹出弹框，填入新增菜单组名创建菜单组

# 2. 菜单组排序

点击 菜单组排序 ，弹出排序弹框，拖动菜单组进行排序

# 3. 编辑菜单组

可进行菜单组重命名、删除以及新增

3.1 菜单组重命名

点击 重命名 ，弹出弹框，填入新的菜单组名

3.2 删除该菜单组

点击 删除该菜单组 ，弹出确认删除弹框，用户可进行相关操作

3.3 新增菜单

点击 新增菜单 ，可选择新增菜单类型（表单、模块、第三方应用），设置菜单名字、菜单图标、使用范围（范围可选部门、职员、角色）、是否禁用

注意

第三方应用必须设置外部链接
