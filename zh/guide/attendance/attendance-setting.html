# 考勤设置

在这个菜单下可进行考勤组管理、班次管理，设定加班规则、补卡规则以及假期管理

# 1. 考勤组管理

注：在经行考勤组管理设置前请先去班次管理中添加班次。

在考勤组管理中可新增考勤组、编辑排班（只有类型为排班制才可编辑排班）、修改考勤规则、删除考勤组，，如下图：

# 新增考勤组

点击 新增考勤组 ，填入相应信息新增考勤组

# 修改规则

点击 修改规则 ，可修改排班规则

# 删除考勤组

点击 删除 ，弹出确认删除弹框，执行相关操作

# 编辑排班

点击 编辑排班 ，进行排班管理

注意

考勤组类型为排班制才可编辑排班

# 2. 班次管理

班次管理中可新增班次、编辑班次、删除班次和班次列表查看,点击班次管理首先会看到班次列表，如下图：

# 2.1 新增班次

点击 班次列表顶部左上角新增班次按钮，右侧会弹出新增班次界面，如下图：

在新增班次界面填写信息点击右下角确定按钮添加一个新班次， 其中班次名字、上/下班时间、班次负责人为必填，打卡范围限制、休息时间可不填。如果填写了打卡范围限制，就只允许在你填写的时间范围内打卡，其他时间段不能打卡，如果填写了休息时间，在计算某天的工作时长会扣除这部分时间。

# 2.2 编辑班次

点击 班次列表的列表右侧编辑按钮 ，右侧会弹出编辑班次界面，可编辑班次信息，编辑内容和新增内容相同。

# 2.3 删除班次

点击 班次列表的列表右侧删除按钮 ，弹出确认删除弹框，点击确认会删除这条班次信息，当班次信息被考勤组使用后不允许删除。

# 2.4 列表搜索查询

点击 班次列表顶部右上角搜索图标，输入关键字（班次名称），可进行筛选查询。

# 3. 加班规则

可新增、编辑、删除加班规则

# 新增加班规则

点击 新增加班规则 ，填入信息

# 编辑加班规则

点击 编辑 ，修改加班规则信息

# 删除加班规则

点击 删除 ，弹出确认删除弹框，执行相关操作

# 4. 补卡规则

可新增、编辑、删除补卡规则

# 新增补卡规则

点击 新增补卡规则 ，填入信息

# 编辑补卡规则

点击 编辑 ，修改补卡规则信息

# 删除补卡规则

点击 删除 ，弹出确认删除弹框，执行相关操作

# 5. 假期管理

详情请见假期管理
